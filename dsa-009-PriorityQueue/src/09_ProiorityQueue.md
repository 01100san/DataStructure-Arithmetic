### 优先队列(Priority Queue)
优先级队列也是个队列，因此也是提供以下接口  
int size();     // 元素的数量  
boolean isEmpty();  // 是否为空  
void enQueue(E element);    // 入对  
E deQueue();    // 出队  
E front(); // 获取队列的头元素  
void clear (); //清空
普通的队列是 FIFO 原则，也就是先进先出  
优先级队列则是按照优先级高低进行出队,比如将优先级最高的元素作为队头优先出队  
<p>优先级队列的应用场景举例：</p>  
医院的夜间门诊,队列元素是病人,口优先级是病情的严重情况、挂号时间<br>  
操作系统的多任务调度,队列元素是任务,优先级是任务类型
